# NASA Habitability Engine (v2.4.0)
**A Multi-Disciplinary Scientific Modeling Tool**

### 🚀 Overview
This project is an interdisciplinary engine designed to predict the biological viability of exoplanets using real-world data from the **NASA Exoplanet Archive**. Originally developed as a **C++11** console application during my HackerRank certification sprint, it has been ported to a web-based interface for public accessibility.

### 🧪 Integrated Sciences
This engine calculates a "Habitability Index" by synthesizing three core scientific domains:

* **Physics:** Implements the **Inverse Square Law** to determine Stellar Flux ($L / d^2$) based on stellar luminosity and orbital distance.
* **Chemistry:** Models the **Greenhouse Effect** by calculating how atmospheric $CO_2$ concentrations trap heat and alter surface temperatures.
* **Biology:** Evaluates survival thresholds (0°C to 100°C) required for liquid water stability and cellular protein integrity.
